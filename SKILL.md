# Swiftly Learning
Whenever the user intends to learn and remember a new concept, use this skill.

**Keywords**: learning,flashcards,concepts

## Procedure
1. Update the LEARNINGS.md file in the project. If the file does not exist, create it at the root of the project.
2. Insert the date and time when writing into the log, following Australian time format (Sydney) at DD-MM-YYYY:HH-MM
3. Within the same line as the date, provide sufficient details on the concept learned. Include the following information:
- Concept: What has the user learned?
- Keywords: Relevant keywords related to the technologies, libraries, frameworks related to the LEARNINGS
